# Magic Pen

A creative Flutter drawing application that lets you draw with a glowing "magic pen". The app uses on-screen touch gestures to provide an intuitive and fluid creative experience.

## Features
- **Magic Pen Glow**: Smooth, glowing strokes that simulate drawing with light.
- **Multi-Touch Gestures**: 
  - One finger to draw.
  - Two-finger tap anywhere on the screen to change the pen color.
  - Three-finger tap to clear the canvas.
- **Cross-Platform**: Fully responsive and works smoothly on mobile, tablet, and desktop environments.

## Setup and Run
1. Ensure Flutter is installed.
2. Run `flutter pub get` to fetch dependencies.
3. Use `flutter run` to launch the app on your preferred platform.
